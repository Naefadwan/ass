Accessibility Statement

Last Updated: May 12, 2025

Our Commitment to Accessibility

E-Pharma is committed to ensuring digital accessibility for people with disabilities. We are continually improving the user experience for everyone and applying the relevant accessibility standards to make our website and mobile application accessible to all users, regardless of ability.

Conformance Status

The Web Content Accessibility Guidelines (WCAG) define requirements for designers and developers to improve accessibility for people with disabilities. It defines three levels of conformance: Level A, Level AA, and Level AAA.

E-Pharma's website is partially conformant with WCAG 2.1 level AA. Partially conformant means that some parts of the content do not fully conform to the accessibility standard.

Compatibility with Browsers and Assistive Technology

E-Pharma's website is designed to be compatible with the following assistive technologies:

  • Screen readers (including JAWS, NVDA, VoiceOver, and TalkBack)
  • Speech recognition software
  • Screen magnification software
  • Keyboard-only navigation

Our website is compatible with recent versions of:

  • Google Chrome
  • Mozilla Firefox
  • Apple Safari
  • Microsoft Edge

Technical Specifications

Accessibility of E-Pharma's website relies on the following technologies to work with the particular combination of web browser and any assistive technologies or plugins installed on your computer:

  • HTML
  • WAI-ARIA
  • CSS
  • JavaScript

These technologies are relied upon for conformance with the accessibility standards used.

Assessment Methods

E-Pharma has assessed the accessibility of our website by the following methods:

  • Self-evaluation
  • External evaluation by accessibility experts
  • User testing with people who use assistive technologies
  • Automated testing tools

Specific Features

We have implemented the following specific accessibility features on our website:

  • Text alternatives for non-text content
  • Captions and other alternatives for multimedia
  • Content that can be presented in different ways without losing information
  • Content that is easier to see and hear
  • Functionality that is available from a keyboard
  • Content that does not cause seizures or physical reactions
  • Navigable content that helps users navigate and find content
  • Readable and understandable text content
  • Content that appears and operates in predictable ways
  • Input assistance to help users avoid and correct mistakes
  • Compatible content that works with current and future user tools

Known Limitations

Despite our best efforts to ensure accessibility of E-Pharma's website, there may be some limitations. Below is a description of known limitations, and potential solutions. Please contact us if you observe an issue not listed below.

  • Interactive prescription labels: Some interactive medication labels may not be fully accessible to screen readers. We are working to improve this feature. In the meantime, please contact our customer service for assistance with prescription information.
  • PDF documents: Some older PDF documents may not be fully accessible. We are working to remediate these documents. If you encounter a PDF document that is not accessible, please contact us for assistance.
  • Video content: Some older video content may not have captions or audio descriptions. We are working to update these videos. If you encounter a video without captions, please contact us for assistance.
