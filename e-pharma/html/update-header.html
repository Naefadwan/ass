Add Headers to All Pages

Instructions

The simplest way to add the header and authentication scripts to all pages:

  1. Copy the code snippet below
  2. Paste it into the console of each HTML page you want to update
  3. After running, save the page to preserve the changes

Code to Copy:


// Add header and authentication scripts to page
(function() {
  // Scripts to add
  const scripts = [
    { src: '../JS/header-fix.js', id: 'header-fix-script' },
    { src: '../JS/user-auth-final.js', id: 'user-auth-final-script' }
  ];
  
  // Add each script to head
  scripts.forEach(script => {
    // Only add if not already present
    if (!document.querySelector(`script[src="${script.src}"]`)) {
      const scriptElem = document.createElement('script');
      scriptElem.src = script.src;
      scriptElem.id = script.id;
      document.head.appendChild(scriptElem);
      console.log(`Added ${script.src}`);
    }
  });
  
  console.log('Header update complete');
})();

Important!

After running this code on each page, you must save the changes to the HTML file.

This ensures the scripts will be loaded when the page is opened next time.
